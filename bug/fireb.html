Firebase Misconfig

Sensitive Data Exposure  

After downloading the APK, I proceeded to decompile it using Jadx. During the decompilation process, I navigated to the strings file, where I discovered a Firebase URL.

<string name="firebase_database_url">https://XXXX-XXXX-application.firebaseio.com</string>

On opening the link, on browser it was showing sensitive data Like Coupon code, email, transaction details etc


A critical security vulnerability arises when users can access the /.json file of a Firebase database, leading to unauthorized data disclosure. This occurs due to misconfigurations in Firebase security rules, allowing unauthenticated or unauthorized users to read the entire database. By accessing the /.json endpoint, attackers can retrieve sensitive information, potentially exposing user data, authentication tokens, or other confidential details.
